# Validate Skill

Evidence-first validation for concrete artifacts, diffs, bug fixes, UI work, plans, and release candidates.

`validate` proves work without delivering it. `finish-task` should call this before review gates and git delivery.

## Installation

```bash
ln -sfn ~/Developer/Personal/skill-validate/skills/validate ~/.claude/skills/validate
ln -sfn ~/Developer/Personal/skill-validate/skills/validate ~/.codex/skills/validate
```
